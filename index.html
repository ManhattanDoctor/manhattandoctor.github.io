Enterprise-платформы

Полный цикл разработки сложных бизнес-систем. Мультитенантность, ресурсная авторизация через Keycloak, двойное хранение состояния (SQL + blockchain), Command/Event архитектура, WebSocket real-time.

AI/ML-платформы

Интеграция LLM в production-системы: LangChain, Claude, OpenAI, векторный поиск в базах данных. Разработка MCP-серверов для AI-агентов. RAG-пайплайны для обработки документов. Гибридный текстовый и семантический поиск.

Криптография и безопасность

Системы на базе российской (ГОСТ Р 34.10) и международной криптографии (RSA, AES, Ed25519). Custody-системы для управления ключами, двухфакторная аутентификация, биометрическая аутентификация с blockchain-верификацией.

Разработка фреймворков и экосистем

Создание переиспользуемых библиотек и экосистемы (30+ npm-пакетов) — инфраструктурная основа для десятка проектов: backend, frontend, криптография, WebSocket, OAuth, i18n. Фреймворк ORM для блокчейна с управлением токенами и сущностями.

Реализованные проекты

DFAWL — White Label конструктор платформ ЦФА / УЦП

12+ компонентов

Конструктор для быстрого развёртывания платформ торговли цифровыми финансовыми активами и утилитарными цифровыми правами в рамках российского законодательства. Двойное хранение состояния — PostgreSQL для оперативных запросов и Hyperledger Fabric для неизменяемого аудита. Мультитенантная архитектура, ресурсная авторизация, блокчейн-эксплорер, AI-модуль, кастодиальный сервис, RESTful API, торговые роботы.

Karma — благотворительная blockchain-платформа

Hyperledger Labs · IEEE ICBC 2021

Первая в России блокчейн-платформа для пожертвований на базе Hyperledger Fabric. Запущена в 2023 году, на платформе работают фонды «Со-единение», «Найди семью», «Большая перемена», «Синдром любви», «Арифметика добра». Доноры отслеживают движение средств в реальном времени при отсутствии комиссии платформы. Принята в Hyperledger Labs (Linux Foundation), представлена на Global Hyperledger Forum 2022, опубликована в IEEE в соавторстве с IBM.

Gnosis — масштабируемая AI-платформа

AI-платформа с мультипровайдерным переключением языковых моделей. Реализует RAG-пайплайн для работы с различными документами, генерацию изображений, синтез речи, распознавание голоса и создание видео-аватаров. Встроенная монетизация и OAuth-аутентификация (Google, Yandex, VK, Telegram).

Cvartel — биометрическая аутентификация

IEEE ICDCSW 2025

Система биометрической аутентификации с верификацией через Hyperledger Fabric. Совместная работа лаборатории ChainLab и МФТИ. Результаты представлены на 45th IEEE International Conference on Distributed Computing Systems Workshops 2025 в Глазго.

DAO (включая Custody, HLF Explorer и Solidity-контракты)

Платформа децентрализованных автономных организаций с токен-голосованием и Ethereum-мостом. Custody-система для управления криптографическими ключами. Обозреватель блокчейна Hyperledger Fabric. Коллекция смарт-контрактов: Security Tokens (Polymath), ERC-20, ERC-721, DEX.

@ts-core — экосистема TypeScript-библиотек

30+ npm-пакетов

Инфраструктурная основа для всех enterprise-проектов: backend и frontend утилиты, криптография (ГОСТ, RSA, MetaMask), WebSocket клиент/сервер, OAuth/OpenID, двухфакторная аутентификация, интернационализация, Swagger-генерация.

@hlf-core — фреймворк для Hyperledger Fabric

10 модулей, 5 npm-пакетов

ORM-подобная абстракция над блокчейном: EntityManager для CRUD-операций, каскадные связи между сущностями, пагинация, строковая арифметика для точных финансовых расчётов. Система управления токенами с эмиссией, переводами, hold-механизмом и полным аудитом.

Vichatter — социальная сеть для видеообщения

>3 млн пользователей и до 500K визитов/день
