Области экспертизы

Blockchain-архитектура

Проектирование систем на Hyperledger Fabric и Ethereum: смарт-контракты, токеномика, кросс-чейн взаимодействие. Автор ORM-фреймворка для chaincode и библиотеки управления токенами. Опыт с Polymath Security Tokens (ST-20), ERC-20/ERC-721, DeFi-механиками.

Enterprise-платформы

Полный цикл разработки сложных бизнес-систем. Мультитенантность, ресурсная авторизация через Keycloak, двойное хранение состояния (SQL + blockchain), Command/Event архитектура, WebSocket real-time. Платформа цифровых финансовых активов из 12+ компонентов — крупнейший реализованный проект.

AI/ML-платформы

Интеграция LLM в production-системы: LangChain, Claude, OpenAI, векторный поиск через Pinecone. Разработка MCP-серверов для AI-агентов. RAG-пайплайны для обработки документов. Гибридный текстовый и семантический поиск.

Криптография и безопасность

Системы на базе российской (ГОСТ Р 34.10) и международной криптографии (RSA, AES, Ed25519). Custody-системы для управления ключами, двухфакторная аутентификация, биометрическая аутентификация с blockchain-верификацией (тема IEEE-публикации 2025).

Разработка фреймворков и экосистем

Создание переиспользуемых библиотек. Экосистема ts-core (30+ npm-пакетов) — инфраструктурная основа для десятка проектов: backend, frontend, криптография, WebSocket, OAuth, i18n. Фреймворк hlf-core — ORM для блокчейна с управлением токенами и сущностями.

Реализованные проекты

DFA — платформа цифровых финансовых активов

12+ компонентов

Полный жизненный цикл ЦФА: выпуск, обращение, OTC-торговля, краудсейлы, погашение. Двойное хранение состояния — PostgreSQL для запросов и Hyperledger Fabric для неизменяемого аудита. Мультитенантная архитектура с ресурсной авторизацией. Backend, frontend, chaincode, AI-интеграция, аутентификация, промо, банковские сервисы.

ts-core — экосистема TypeScript-библиотек

30+ npm-пакетов

Инфраструктурная основа для всех enterprise-проектов: backend и frontend утилиты, криптография (ГОСТ, RSA, MetaMask), WebSocket клиент/сервер, OAuth/OpenID, двухфакторная аутентификация, интернационализация, Swagger-генерация.

hlf-core — фреймворк для Hyperledger Fabric

10 модулей, 5 npm-пакетов

ORM-подобная абстракция над блокчейном: EntityManager для CRUD-операций, каскадные связи между сущностями, пагинация, строковая арифметика для точных финансовых расчётов. Система управления токенами с эмиссией, переводами, hold-механизмом и полным аудитом.

Karma — благотворительная blockchain-платформа

Hyperledger Labs, IEEE ICBC 2021

Платформа для благотворительных фондов, обеспечивающая прозрачность сбора и расходования пожертвований через неизменяемый blockchain-аудит. Принята в Hyperledger Labs (Linux Foundation). Научная публикация в соавторстве с IBM.

Gnosis — AI-платформа

Мультимодельный AI-ассистент с анализом документов, семантическим поиском по векторной базе данных и интерактивным чатом. Обрабатывает DOCX, PDF, изображения. Доступен через веб-интерфейс и Telegram.

Cvartel — биометрическая аутентификация

IEEE ICBC 2025

Система биометрической аутентификации с верификацией через Hyperledger Fabric. Результаты представлены на IEEE International Conference on Blockchain and Cryptocurrency 2025 в Пизе.

DAO, Custody, HLF Explorer, Solidity-контракты

Платформа децентрализованных автономных организаций с токен-голосованием и Ethereum-мостом. Custody-система для управления криптографическими ключами. Обозреватель блокчейна Hyperledger Fabric. Коллекция смарт-контрактов: Security Tokens (Polymath), ERC-20, ERC-721, DEX.

Vichatter — социальная сеть для видеообщения

>3 млн пользователей, до 500K визитов/день

Международная видео-социальная сеть с live-стримингом, встроенным переводчиком сообщений, системой виртуальных подарков и рейтингов, кастомным 3D-движком. Мобильные и десктопные приложения. Партнёрство с PROMT и Лигой безопасного интернета.
